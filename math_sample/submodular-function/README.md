# submodular function simulation

このサンプルは、劣モジュラー関数（facility-location）を使って、
「少数の代表点で全体を広くカバーする」選択問題をシミュレーションします。

## setup

```shell
pip install numpy matplotlib pillow
```

## what this simulation represents

このシミュレーションは、次のような実世界の問題設定を想定しています。

- レコメンドの多様化: 上位 k 件を似た候補で埋めず、カテゴリ全体をカバーする
- 検索結果の多様化: 同一意図に偏らない結果セットを返す
- 要約・ハイライト抽出: 重複を抑えながら情報カバレッジを高める
- 施設配置: 限られた拠点数で需要点全体への到達性を上げる
- センサー配置: 少数センサーで監視対象を広くカバーする

## mapping to the code

- 点群 `x`: 候補アイテム（商品、文、地点など）
- 類似度行列 `sim`: 候補間の近さ（RBF カーネルで作るグラム行列）
- 目的関数 `F(S) = sum_i max_{j in S} sim(i, j)`: 選択集合 `S` による全体カバレッジ
- 貪欲法: 各ステップで限界利得が最大の候補を追加

このため本サンプルは、「限られた枠で重複を減らしつつ代表性を高める」問題の基本形になっています。
